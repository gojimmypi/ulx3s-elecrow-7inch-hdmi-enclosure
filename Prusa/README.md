## G-Code Files

The G-Code files were generated for a Prusa MK3S+ printer.

### Prusa Slicer Main Body and Print Set G-Code

When printing, the two enclosure halves and the small parts set can be used instead of printing each part individually.

The Adapter Base Plat Set is included in the Small Parts and can also be printed separately as a set.

- Adapter Base Plate Set_0.15mm_PETG_MK3S_1h21m.gcode
- Enclosure Display Side_0.15mm_PETG_MK3S_7h5m.gcode
- Enclosure ULX3S Side_0.15mm_PETG_MK3S_8h55m.gcode
- Small parts_0.15mm_PETG_MK3S_5h5m.gcode
